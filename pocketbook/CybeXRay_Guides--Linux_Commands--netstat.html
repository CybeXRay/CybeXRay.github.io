netstat


netstat command
netstat : Gather information about existing communications
netstat -a : Shows all listening ports and established connections
netstat -at : Shows all TCP
netstat -au : Shows all UDP
netstat -l : Shows ports in listening mode. (Ready for connection) . Can be used as -lt or -lu for tcp & udp
netstat -s : Shows network usage statistics. Can be used as -st or -su for tcp & udp
netstat -tp : Shows connections with the service name and PID information. (TCP) [Can be used with -l (Listening)]
netstat -up : Shows connections with the service name and PID information. (UDP) [Can be used with -l (Listening)]
netstat -i : shows interface statistics

netstat -ano : Used in many netstat guies
-a: Display all sockets
-n: Do not resolve names
-o: Display timers
