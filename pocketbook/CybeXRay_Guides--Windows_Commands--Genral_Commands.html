Genral Commands


Command Line/Run:

Local User and Group Management : lusrmgr.msc
System Configuration : msconfig.exe
Task Manager Command : taskmgr.exe
Command for Troubleshooting : C:\Windows\System32\control.exe /name Microsoft.Troubleshooting
Control Panel Command : control.exe
UAC Settings : UserAccountControlSettings.exe
Computer Management : compmgmt.msc
Performance Monitor : perfmon.exe
System Information : msinfo32.exe
Resource Monitor : resmon.exe
Registry Editor : regedt32.exe & regedit.exe




tasklist /m /fi "pid eq 1304" : Command to list all the dll files used by a certain pid.


net user <username> <password> /add : Add an user in command line
net user <username> <new_password> : To change password of username
net localgroup administrators <username> /add : Giving admin rights to a user
