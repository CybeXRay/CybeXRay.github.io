Network Scanning


RDP to Windows machine


xfreerdp /v:ip_address /u:username /p:password

Options:
/cert:ignore --→ Ignores the certificate to avoid any error
/f --→ Full screen mode
/dynamic-resolution --→ Better than full screen. The resolution changes according to window size.
+clipboard --→ Used to copy paste clipboard items between Kali and connected Windows Machine.


eg.
xfreerdp /dynamic-resolution +clipboard /cert:ignore /v:10.10.229.129 /u:administrator /p:4q6XvFES7Fdxs
