registry commands


To Change Registry Setting From Command Line:



Note: Registry and GPEdit are complimentary, thus changes are reflected in both of them.

Reference: https://www.windowscentral.com/how-edit-registry-using-command-prompt-windows-10



Eg. Adding and Deleteing Registry Key

REG ADD "\\RetroWeb\HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWORD /d 1

REG DELETE "\\RetroWeb\HKLM\SOFTWARE\Policies\Microsoft\Windows Defender"


For Local Machine, we can use hostname.


REG ADD "\\<hostname>\HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWORD /d 1

REG DELETE "\\<hostname>\HKLM\SOFTWARE\Policies\Microsoft\Windows Defender"


Or

REG ADD "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWORD /d 1

REG DELETE "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender"
