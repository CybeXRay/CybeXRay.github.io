BgInfo

It automatically displays relevant information about a Windows computer on the desktop's background, such as the computer name, IP address, service pack version, and more.

This is a handy utility if you manage multiple machines. This tool, or similar tools, are typically utilized on servers. When a user RDPs into a server, the system information is displayed on the wallpaper to provide quick information about the server, such as the server's name.


Command: Bginfo.exe
Note: After program starts, select the components necessary, then we can preview or save it, To show on Desktop.


RegJump

This little command-line applet takes a registry path and makes Regedit open to that path. It accepts root keys in standard (e.g. HKEY_LOCAL_MACHINE) and abbreviated form (e.g. HKLM).

When navigating through the registry using the Registry Editor, one must manually drill down to the key you wish to inspect.

Command: regjump.exe REG_PATH

Alternatives:
There are multiple ways to query the Windows Registry without using the Registry Editor, such as via the CMD (reg query) and PowerShell (Get-Item/Get-ItemProperty).

Note: Using Regjump will open the Registry Editor and automatically open the editor directly at the path, so one doesn't need to navigate it manually.


Strings

Strings just scans the file you pass it for UNICODE (or ASCII) strings of a default length of 3 or more UNICODE (or ASCII) characters. Note that it works under Windows 95 as well.

Command: strings.exe FILE_NAME
