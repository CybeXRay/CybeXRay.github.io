SQLITE

We will be using sqlite3 command line tool in Kali to Enumerate a flat-file database.

Command To Open The Database:

sqlite3 db_name.db

Sqlite Commands:

.tables : List All the Tables in the Database

.quit : Exit Sqlite3

PRAGMA table_info(<Table_Name>); : Gives Information about a particular table.

SELECT * FROM <Table_Name>; : Show the contents of a particular table.

.schema <table_name> : Shows the table schema

UPDATE <table_name> SET <column_name>=<value> WHERE <column_name>=<conditional_value>; : Updating Table
