ProFTPd (Vulnerable)


Version 1.3.5

The above version is vulnearble.

There is a exploit in ProFtpd's mod_copy module.

The mod_copy module implements SITE CPFR and SITE CPTO commands, which can be used to copy files/directories from one place to another on the server. Any unauthenticated client can leverage these commands to copy files from any part of the filesystem to a chosen destination.

SITE CPFR : Copy From Path

SITE CPTO : Copy To Path

Any unauthenticated user can use the above commands to copy files after connecting to the ProFTPd server.

Process to Connect:

nc <Server_IP> <Port>
