xfreerdp

RDP to Windows machine

xfreerdp /v:ip_address /u:username /p:password

Options:

/cert:ignore --→ Ignores the certificate to avoid any error

/f --→ Full screen mode

/dynamic-resolution --→ Better than full screen. The resolution changes according to window size.

+clipboard --→ Used to copy paste clipboard items between Kali and connected Windows Machine.

/scale --→ Used to scale windows size

+home-drive --→ To share the home directory of local user as a network share in the remote machine

/drive:drive_name,drive_path --→ Used to share a local directory at the drive_path to be shared in network as drive_name to the remote machine

eg.

xfreerdp /dynamic-resolution +clipboard /cert:ignore /v:10.10.229.129 /u:administrator /p:4q6XvFES7Fdxs

xfreerdp /dynamic-resolution +clipboard /scale:140 /cert:ignore /v:10.10.56.226 /u:mayor /p:'8CL7O1N78MdrCIsV'

xfreerdp /f +clipboard /cert:ignore /scale:140 /v:10.10.226.27 /u:admin /p:password

xfreerdp /dynamic-resolution +clipboard +home-drive /cert:ignore /scale:140 /v:10.10.144.167 /u:thm-unpriv /p:Password321

xfreerdp /f +clipboard /drive:CybeXRay,/home/cybex/lab /cert:ignore /scale:140 /v:10.10.144.167 /u:thm-unpriv /p:Password321

Domain Connections

eg.

xfreerdp -f /v:10.10.103.12 /u:spookysec.local\svc-admin /cert:ignore /p:management2005
