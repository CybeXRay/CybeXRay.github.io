ss


Socket Status

Similar to Netstat For cases Where Netstat is not installed.

ss

man ss For more details

Examples:

ss -a or ss --all :: To List All Connections (Listening & Established)

ss -l :: Ro List All Listening Connections

ss -t or ss --tcp :: To List All TCP Connections

ss -lt :: To List All Listening TCP Connections

To show all Open Ports:

ss -tulwn
