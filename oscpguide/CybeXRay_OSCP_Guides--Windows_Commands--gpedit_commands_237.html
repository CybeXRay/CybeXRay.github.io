GPEdit Commands Using Powershell:

Note: Registry and GPEdit are complimentary, thus changes are reflected in both of them.

Note: GPEdit doesn't have command line settings, thus we ideally change the corresponding registry key.

Searching the Corresponding Registry key for a GPEdit Option:

Reference: https://gpsearch.azurewebsites.net/

Either search in the search box or select from the components in the left. Once the corresponding registry key is found. We can use the following set & view it in Powershell

Reference: https://docs.microsoft.com/en-us/powershell/module/grouppolicy/?view=windowsserver2022-ps

Get-GPRegistryValue

Set-GPRegistryValue

Eg.

Get-GPRegistryValue -Name TestGPO -Key "HKEY_CURRENT_USER\Software\Policies\Microsoft\ExampleKey" -ValueName "ValueOne"

Set-GPRegistryValue -Name "TestGPO" -Key "HKCU\Software\Policies\Microsoft\Windows\Control Panel\Desktop" -ValueName "ScreenSaveTimeOut" -Type DWORD -Value 900
