windows defender


Disabling Windows Defender:

(After Having Administrator Privileges)

Checking the Service:

To Check about the service

sc qc windefend

To check Running Status

sc query windefend

Different Version of Windows have different Protection Mechanisms.

Tamper Protection

If tamper protection is disabled in Windows Defender GUI. The we can use the following powershell commands to disable and enable real time protection.

To Disable

Set-MpPreference -DisableRealtimeMonitoring $true

To Enable

Set-MpPreference -DisableRealtimeMonitoring $false

Tamper Protection Service(MsMpEng.exe)

Another way to deal with tamper protection is to Suspend the MsMpEng.exe process.

Note: We cannot stop the process even from Administrator account

Suspending From Task Manager: Head over to Task Manager, then to Performance Monitor from the Performance tab. Find MsMpEng.exe in any of the tabs in it (mostly on top in Memory or CPU tabs) and click on Suspend Process.

Note: It is causing some system instability on some updated windows (Windows 10 Enterprise)

GPEdit Method (Tested with Older Windows Server 2016/2019):

Open GPEdit.msc as administrator. Then go to Computer Configuration ---> Administrative Templates ---> Windows Components ---> Windows Defender

Note: The location varies just a little for different version of windows

The main key to be changed: Turn off Windows Defender = Enabled

Registry Method (Corresponding Registry Entry for the above GPEdit Option):

Key: HKLM\SOFTWARE\Policies\Microsoft\Windows Defender

Valuename: DisableAntiSpyware

Value: 01

Note: In Some systems (Windows Server 2016) the Registry method changes the registry but windows defender keeps on running. (Reboot Required). However, using GPEdit method the defender was successfully disabled (Windows Server 2016)

However: In Latest Windows 10 Enterprise (Updated), both the methods do not work.
